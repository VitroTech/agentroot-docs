GitHub through AgentRoot. Your agent never holds the GitHub API key — every call routes through the proxy, the key is injected from the vault, and the call is classified, gated, and audited.

## Setup

1. Dashboard → **Connect** → GitHub
2. Paste your GitHub API key into the secure iframe
3. Bind the key to one or more agents
4. Point your code at `https://proxy.agentroot.app/github`

For the conventions every example assumes — required headers, error codes, idempotency — see [Conventions](../conventions).

## Action class

GitHub endpoints classify as **Writes** by default. See [Action Classes](/action-classes) for what governs this class and the relevant per-class auto-kill thresholds. To override per-route, see [Action Class Mapping](../action-class-mapping#overriding-classifications).

## Calling GitHub through AgentRoot

Representative call — `POST /repos/{owner}/{repo}/issues`:

<CodeGroup>
```bash cURL
curl -X POST 'https://proxy.agentroot.app/github/repos/{owner}/{repo}/issues' \
      -H "Authorization: Bearer $AGENTROOT_TOKEN" \
      -H "X-AgentRoot-ID: $AGENTROOT_AGENT_ID" \
      -H "Content-Type: application/json" \
      -d '{
  "title": "From the agent",
  "body": "This issue was filed by an AgentRoot-governed agent."
}'
```

```typescript TypeScript
await fetch('https://proxy.agentroot.app/github/repos/{owner}/{repo}/issues', {
  method: 'POST',
  headers: {
    Authorization: `Bearer ${process.env.AGENTROOT_TOKEN}`,
    'X-AgentRoot-ID': process.env.AGENTROOT_AGENT_ID!,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
  "title": "From the agent",
  "body": "This issue was filed by an AgentRoot-governed agent."
}),
});
```

```python Python
import os, requests
requests.post(
      "https://proxy.agentroot.app/github/repos/{owner}/{repo}/issues",
      headers={
        "Authorization": f"Bearer {os.environ['AGENTROOT_TOKEN']}",
        "X-AgentRoot-ID": os.environ["AGENTROOT_AGENT_ID"],
      },
      json={
  "title": "From the agent",
  "body": "This issue was filed by an AgentRoot-governed agent."
},
    )
```
</CodeGroup>
