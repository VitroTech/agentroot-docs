Google Calendar through AgentRoot. Your agent never holds the Google Calendar API key — every call routes through the proxy, the key is injected from the vault, and the call is classified, gated, and audited.

## Setup

1. Dashboard → **Connect** → Google Calendar
2. Paste your Google Calendar API key into the secure iframe
3. Bind the key to one or more agents
4. Point your code at `https://proxy.agentroot.app/google-calendar`

For the conventions every example assumes — required headers, error codes, idempotency — see [Conventions](../conventions).

## Action class

Google Calendar endpoints classify as **Writes** by default. See [Action Classes](/action-classes) for what governs this class and the relevant per-class auto-kill thresholds. To override per-route, see [Action Class Mapping](../action-class-mapping#overriding-classifications).

## Calling Google Calendar through AgentRoot

Representative call — `POST /calendar/v3/calendars/primary/events`:

<CodeGroup>
```bash cURL
curl -X POST 'https://proxy.agentroot.app/google-calendar/calendar/v3/calendars/primary/events' \
      -H "Authorization: Bearer $AGENTROOT_TOKEN" \
      -H "X-AgentRoot-ID: $AGENTROOT_AGENT_ID" \
      -H "Content-Type: application/json" \
      -d '{
  "summary": "Meeting",
  "start": {
    "dateTime": "2026-06-01T10:00:00Z"
  },
  "end": {
    "dateTime": "2026-06-01T11:00:00Z"
  }
}'
```

```typescript TypeScript
await fetch('https://proxy.agentroot.app/google-calendar/calendar/v3/calendars/primary/events', {
  method: 'POST',
  headers: {
    Authorization: `Bearer ${process.env.AGENTROOT_TOKEN}`,
    'X-AgentRoot-ID': process.env.AGENTROOT_AGENT_ID!,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
  "summary": "Meeting",
  "start": {
    "dateTime": "2026-06-01T10:00:00Z"
  },
  "end": {
    "dateTime": "2026-06-01T11:00:00Z"
  }
}),
});
```

```python Python
import os, requests
requests.post(
      "https://proxy.agentroot.app/google-calendar/calendar/v3/calendars/primary/events",

  }
},
    )
```
