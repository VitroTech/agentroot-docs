Twilio through AgentRoot. Your agent never holds the Twilio API key — every call routes through the proxy, the key is injected from the vault, and the call is classified, gated, and audited.

## Setup

1. Dashboard → **Connect** → Twilio
2. Paste your Twilio API key into the secure iframe
3. Bind the key to one or more agents
4. Point your code at `https://proxy.agentroot.app/twilio`

For the conventions every example assumes — required headers, error codes, idempotency — see [Conventions](../conventions).

## Action class

Twilio endpoints classify as **Communication** by default. See [Action Classes](/action-classes) for what governs this class and the relevant per-class auto-kill thresholds. To override per-route, see [Action Class Mapping](../action-class-mapping#overriding-classifications).

## Calling Twilio through AgentRoot

Representative call — `POST /2010-04-01/Accounts/{AccountSid}/Messages.json`:

<CodeGroup>
```bash cURL
curl -X POST 'https://proxy.agentroot.app/twilio/2010-04-01/Accounts/{AccountSid}/Messages.json' \
      -H "Authorization: Bearer $AGENTROOT_TOKEN" \
      -H "X-AgentRoot-ID: $AGENTROOT_AGENT_ID" \
      -H "Content-Type: application/json" \
      -d '{
  "To": "+15551234567",
  "From": "+15559876543",
  "Body": "hello from AgentRoot"
}'
```

```typescript TypeScript
await fetch('https://proxy.agentroot.app/twilio/2010-04-01/Accounts/{AccountSid}/Messages.json', {
  method: 'POST',
  headers: {
    Authorization: `Bearer ${process.env.AGENTROOT_TOKEN}`,
    'X-AgentRoot-ID': process.env.AGENTROOT_AGENT_ID!,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
  "To": "+15551234567",
  "From": "+15559876543",
  "Body": "hello from AgentRoot"
}),
});
```

```python Python
import os, requests
requests.post(
      "https://proxy.agentroot.app/twilio/2010-04-01/Accounts/{AccountSid}/Messages.json",
      headers={
        "Authorization": f"Bearer {os.environ['AGENTROOT_TOKEN']}",
        "X-AgentRoot-ID": os.environ["AGENTROOT_AGENT_ID"],
      },
      json={
  "To": "+15551234567",
  "From": "+15559876543",
  "Body": "hello from AgentRoot"
},
    )
```
</CodeGroup>
