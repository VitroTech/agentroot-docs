Cartesia through AgentRoot. Your agent never holds the Cartesia API key — every call routes through the proxy, the key is injected from the vault, and the call is classified, gated, and audited.

## Setup

1. Dashboard → **Connect** → Cartesia
2. Paste your Cartesia API key into the secure iframe
3. Bind the key to one or more agents
4. Point your code at `https://proxy.agentroot.app/cartesia`

For the conventions every example assumes — required headers, error codes, idempotency — see [Conventions](../conventions).

## Action class

Cartesia endpoints classify as **Spend** by default. See [Action Classes](/action-classes) for what governs this class and the relevant per-class auto-kill thresholds. To override per-route, see [Action Class Mapping](../action-class-mapping#overriding-classifications).

## Calling Cartesia through AgentRoot

Representative call — `POST /tts/bytes`:

<CodeGroup>
```bash cURL
curl -X POST 'https://proxy.agentroot.app/cartesia/tts/bytes' \
      -H "Authorization: Bearer $AGENTROOT_TOKEN" \
      -H "X-AgentRoot-ID: $AGENTROOT_AGENT_ID" \
      -H "Content-Type: application/json" \
      -d '{
  "transcript": "Hello from AgentRoot",
  "model_id": "sonic-english",
  "voice": {
    "mode": "id",
    "id": "<voice_id>"
  }
}'
```

```typescript TypeScript
await fetch('https://proxy.agentroot.app/cartesia/tts/bytes', {
  method: 'POST',
  headers: {
    Authorization: `Bearer ${process.env.AGENTROOT_TOKEN}`,
    'X-AgentRoot-ID': process.env.AGENTROOT_AGENT_ID!,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
  "transcript": "Hello from AgentRoot",
  "model_id": "sonic-english",
  "voice": {
    "mode": "id",
    "id": "<voice_id>"
  }
}),
});
```

```python Python
import os, requests
requests.post(
      "https://proxy.agentroot.app/cartesia/tts/bytes",
      headers={
        "Authorization": f"Bearer {os.environ['AGENTROOT_TOKEN']}",
        "X-AgentRoot-ID": os.environ["AGENTROOT_AGENT_ID"],
      },
      json={
  "transcript": "Hello from AgentRoot",
  "model_id": "sonic-english",
  "voice": {
    "mode": "id",
    "id": "<voice_id>"
  }
},
    )
```
</CodeGroup>
