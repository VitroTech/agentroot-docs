xAI (Grok) through AgentRoot. Your agent never holds the xAI (Grok) API key — every call routes through the proxy, the key is injected from the vault, and the call is classified, gated, and audited.

## Setup

1. Dashboard → **Connect** → xAI (Grok)
2. Paste your xAI (Grok) API key into the secure iframe
3. Bind the key to one or more agents
4. Point your code at `https://proxy.agentroot.app/xai`

For the conventions every example assumes — required headers, error codes, idempotency — see [Conventions](../conventions).

## Action class

xAI (Grok) endpoints classify as **Spend** by default. See [Action Classes](/action-classes) for what governs this class and the relevant per-class auto-kill thresholds. To override per-route, see [Action Class Mapping](../action-class-mapping#overriding-classifications).

## Calling xAI (Grok) through AgentRoot

Representative call — `POST /v1/chat/completions`:

<CodeGroup>
```bash cURL
curl -X POST 'https://proxy.agentroot.app/xai/v1/chat/completions' \
      -H "Authorization: Bearer $AGENTROOT_TOKEN" \
      -H "X-AgentRoot-ID: $AGENTROOT_AGENT_ID" \
      -H "Content-Type: application/json" \
      -d '{
  "model": "grok-2",
  "messages": [
    {
      "role": "user",
      "content": "Hello"
    }
  ]
}'
```

```typescript TypeScript
await ar.proxy({
      provider: 'xai',
      path: '/v1/chat/completions',
      method: 'POST',,
      body: {
  "model": "grok-2",
  "messages": [
    {
      "role": "user",
      "content": "Hello"
    }
  ]
},
    });
```

```python Python
import os, requests
requests.post(
      "https://proxy.agentroot.app/xai/v1/chat/completions",
      headers={
        "Authorization": f"Bearer {os.environ['AGENTROOT_TOKEN']}",
        "X-AgentRoot-ID": os.environ["AGENTROOT_AGENT_ID"],
      },
      json={
  "model": "grok-2",
  "messages": [
    {
      "role": "user",
      "content": "Hello"
    }
  ]
},
    )
```
</CodeGroup>
