Groq through AgentRoot. Your agent never holds the Groq API key — every call routes through the proxy, the key is injected from the vault, and the call is classified, gated, and audited.

## Setup

1. Dashboard → **Connect** → Groq
2. Paste your Groq API key into the secure iframe
3. Bind the key to one or more agents
4. Point your code at `https://proxy.agentroot.app/groq`

For the conventions every example assumes — required headers, error codes, idempotency — see [Conventions](../conventions).

## Action class

Groq endpoints classify as **Spend** by default. See [Action Classes](/action-classes) for what governs this class and the relevant per-class auto-kill thresholds. To override per-route, see [Action Class Mapping](../action-class-mapping#overriding-classifications).

## Calling Groq through AgentRoot

Representative call — `POST /openai/v1/chat/completions`:

<CodeGroup>
```bash cURL
curl -X POST 'https://proxy.agentroot.app/groq/openai/v1/chat/completions' \
      -H "Authorization: Bearer $AGENTROOT_TOKEN" \
      -H "X-AgentRoot-ID: $AGENTROOT_AGENT_ID" \
      -H "Content-Type: application/json" \
      -d '{
  "model": "llama-3.1-70b-versatile",
  "messages": [
    {
      "role": "user",
      "content": "Hello"
    }
  ]
}'
```

```typescript TypeScript
await fetch('https://proxy.agentroot.app/groq/openai/v1/chat/completions', {
  method: 'POST',
  headers: {
    Authorization: `Bearer ${process.env.AGENTROOT_TOKEN}`,
    'X-AgentRoot-ID': process.env.AGENTROOT_AGENT_ID!,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
  "model": "llama-3.1-70b-versatile",
  "messages": [
    {
      "role": "user",
      "content": "Hello"
    }
  ]
}),
});
```

```python Python
import os, requests
requests.post(
      "https://proxy.agentroot.app/groq/openai/v1/chat/completions",
      headers={
        "Authorization": f"Bearer {os.environ['AGENTROOT_TOKEN']}",
        "X-AgentRoot-ID": os.environ["AGENTROOT_AGENT_ID"],
      },
      json={
  "model": "llama-3.1-70b-versatile",
  "messages": [
    {
      "role": "user",
      "content": "Hello"
    }
  ]
},
    )
```
</CodeGroup>
