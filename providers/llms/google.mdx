Google Gemini through AgentRoot. Your agent never holds the Google Gemini API key — every call routes through the proxy, the key is injected from the vault, and the call is classified, gated, and audited.

## Setup

1. Dashboard → **Connect** → Google Gemini
2. Paste your Google Gemini API key into the secure iframe
3. Bind the key to one or more agents
4. Point your code at `https://proxy.agentroot.app/google`

For the conventions every example assumes — required headers, error codes, idempotency — see [Conventions](../conventions).

## Action class

Google Gemini endpoints classify as **Spend** by default. See [Action Classes](/action-classes) for what governs this class and the relevant per-class auto-kill thresholds. To override per-route, see [Action Class Mapping](../action-class-mapping#overriding-classifications).

## Calling Google Gemini through AgentRoot

Representative call — `POST /v1beta/models/gemini-1.5-pro:generateContent`:

<CodeGroup>
```bash cURL
curl -X POST 'https://proxy.agentroot.app/google/v1beta/models/gemini-1.5-pro:generateContent' \
      -H "Authorization: Bearer $AGENTROOT_TOKEN" \
      -H "X-AgentRoot-ID: $AGENTROOT_AGENT_ID" \
      -H "Content-Type: application/json" \
      -d '{
  "contents": [
    {
      "parts": [
        {
          "text": "Hello"
        }
      ]
    }
  ]
}'
```

```typescript TypeScript
await fetch('https://proxy.agentroot.app/google/v1beta/models/gemini-1.5-pro:generateContent', {
  method: 'POST',
  headers: {
    Authorization: `Bearer ${process.env.AGENTROOT_TOKEN}`,
    'X-AgentRoot-ID': process.env.AGENTROOT_AGENT_ID!,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
  "contents": [
    {
      "parts": [
        {
          "text": "Hello"
        }
      ]
    }
  ]
}),
});
```

```python Python
import os, requests
requests.post(
      "https://proxy.agentroot.app/google/v1beta/models/gemini-1.5-pro:generateContent",
      headers={
        "Authorization": f"Bearer {os.environ['AGENTROOT_TOKEN']}",
        "X-AgentRoot-ID": os.environ["AGENTROOT_AGENT_ID"],
      },
      json={
  "contents": [
    {
      "parts": [
        {
          "text": "Hello"
        }
      ]
    }
  ]
},
    )
```
</CodeGroup>
