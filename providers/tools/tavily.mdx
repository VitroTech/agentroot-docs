Tavily through AgentRoot. Your agent never holds the Tavily API key — every call routes through the proxy, the key is injected from the vault, and the call is classified, gated, and audited.

## Setup

1. Dashboard → **Connect** → Tavily
2. Paste your Tavily API key into the secure iframe
3. Bind the key to one or more agents
4. Point your code at `https://proxy.agentroot.app/tavily`

For the conventions every example assumes — required headers, error codes, idempotency — see [Conventions](../conventions).

## Action class

Tavily endpoints classify as **Writes** by default. See [Action Classes](/action-classes) for what governs this class and the relevant per-class auto-kill thresholds. To override per-route, see [Action Class Mapping](../action-class-mapping#overriding-classifications).

## Calling Tavily through AgentRoot

Representative call — `POST /search`:

<CodeGroup>
```bash cURL
curl -X POST 'https://proxy.agentroot.app/tavily/search' \
      -H "Authorization: Bearer $AGENTROOT_TOKEN" \
      -H "X-AgentRoot-ID: $AGENTROOT_AGENT_ID" \
      -H "Content-Type: application/json" \
      -d '{
  "api_key": "<auto-injected>",
  "query": "latest AgentRoot news"
}'
```

```typescript TypeScript
await fetch('https://proxy.agentroot.app/tavily/search', {
  method: 'POST',
  headers: {
    Authorization: `Bearer ${process.env.AGENTROOT_TOKEN}`,
    'X-AgentRoot-ID': process.env.AGENTROOT_AGENT_ID!,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
  "api_key": "<auto-injected>",
  "query": "latest AgentRoot news"
}),
});
```

```python Python
import os, requests
requests.post(
      "https://proxy.agentroot.app/tavily/search",
      headers={
        "Authorization": f"Bearer {os.environ['AGENTROOT_TOKEN']}",
        "X-AgentRoot-ID": os.environ["AGENTROOT_AGENT_ID"],
      },
      json={
  "api_key": "<auto-injected>",
  "query": "latest AgentRoot news"
},
    )
```
</CodeGroup>
