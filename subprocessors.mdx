# Subprocessors

AgentRoot uses the following third-party services to operate the platform. Each entry includes what they process on our behalf, where data is held, and their current security attestations.

This list is part of our [Trust Center](trust-center). Material changes are announced to customers who have opted into security notifications with at least 30 days' notice.

## Production Infrastructure

### Cloudflare
- **What they process:** HTTPS traffic to AgentRoot (dashboard, API, proxy); Cloudflare Workers execute the secure key-capture and proxy code paths; DNS; DDoS and WAF protection
- **Data:** Request metadata, IP addresses, encrypted payloads in transit. Cloudflare cannot decrypt API keys or vault contents.
- **Region:** Global edge for traffic handling; logs primarily United States
- **Certifications:** SOC 2 Type II, ISO 27001, PCI DSS, FedRAMP Moderate

### HashiCorp Cloud Platform (HCP) — Vault
- **What they process:** Encrypted storage of customer API keys and agent private keys; HSM-backed unseal
- **Data:** Encrypted secrets. Cleartext is never visible to HashiCorp infrastructure due to envelope encryption performed before storage.
- **Region:** US-East (AWS us-east-1 region)
- **Certifications:** SOC 2 Type II, ISO 27001

### Amazon Web Services (AWS)
- **What they process:** AgentRoot application servers, primary database (agent metadata, audit logs, operator account records), and supporting compute and storage. Database is operated as a managed PostgreSQL service.
- **Data:** Agent metadata, operator wallet addresses, audit log entries. No plaintext API keys, no private keys.
- **Region:** US-East (us-east-1) primary; encrypted backups replicated cross-region within the United States
- **Certifications:** SOC 1 Type II, SOC 2 Type II, ISO 27001, ISO 27017, ISO 27018, PCI DSS Level 1, FedRAMP High, HIPAA

## Identity and Onchain

### Ethereum / Base mainnet (via EAS — Ethereum Attestation Service)
- **What they process:** EAS attestations recording agent identity, agent public keys, and revocation events
- **Data:** Public — agent public keys, operator wallet addresses, attestation timestamps. No private keys, no API keys, no PII beyond what the operator has chosen to associate with their wallet.
- **Region:** Public blockchain; operator wallets can be located anywhere
- **Certifications:** N/A (public ledger; trust derives from cryptographic verification rather than vendor attestation)

## Operational

### Stripe
- **What they process:** Billing, subscription management, payment card data
- **Data:** Operator name, billing email, payment method. Card data is held entirely by Stripe — AgentRoot's systems store only the Stripe customer ID and subscription metadata.
- **Region:** Primarily US, with EU presence
- **Certifications:** SOC 1 Type II, SOC 2 Type II, ISO 27001, PCI DSS Level 1

### Resend
- **What they process:** Transactional notifications (security alerts, billing notices, team invitations)
- **Data:** Recipient email address and transactional message content. No agent payloads, no API key material.
- **Region:** United States (primary)
- **Certifications:** SOC 2 Type II, GDPR-compliant data processing

### Sentry
- **What they process:** Application error traces with sensitive fields scrubbed at the source
- **Data:** Stack traces, request metadata. Configured to exclude request/response payloads, vault key material, customer API keys, and PII fields.
- **Region:** United States (primary)
- **Certifications:** SOC 2 Type II, ISO 27001, HIPAA (BAA available), GDPR-compliant

### Product analytics
At launch, AgentRoot does not use a third-party product analytics vendor on customer-facing surfaces. First-party telemetry (page load, feature usage counts) is collected via AgentRoot's own infrastructure and is not shared with third parties. If we add a vendor in this category in the future, we will list them here and notify customers per the change-notification policy below.

## How We Change This List

Material changes — adding a new subprocessor, removing an existing one, or changing the data category that an existing subprocessor processes — are reflected on this page and announced via email to customers who have opted into security change notifications. We provide at least **30 days' notice** before a material change takes effect.

To opt into change notifications: security@agentroot.com.
